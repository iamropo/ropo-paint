# ropo-paint
Electron version of the paint program from Eloquent js

##Instructions

**Clone the Repository**
```
git clone git@github.com:iamropo/ropo-paint.git
```
**Navigate to the ropo-paint directory**
```
cd ropo-paint/
```
**Install the Dependencies**
```
npm install
```
**Start the App**
```
 npm start
```
